# 私のためのアプリ（私の司令塔）

私専用の、完全プライベートなパーソナル・ダッシュボードです。
**自分のPCの中だけで動き、外部には公開しません。** 会社（社員用）のツール群とは完全に別物です。

## 起動のしかた（かんたん）

**`司令塔を起動.app` をダブルクリック** すると、ブラウザが自動で開きます（アドレスは `http://localhost:4823`）。
このアプリはターミナルを使わずに起動するので、確実です。

- 初回だけ「開発元を確認できない」と出たら、`司令塔を起動.app` を**右クリック →「開く」**で許可。
- このMacには通常のNodeが無いため、**Codexアプリ同梱のNode**を自動で使います（追加インストール不要）。
- アプリは**バックグラウンドで動き続けます**。Macを再起動した後などは、また `司令塔を起動.app` を開けばOK。
- Dockに置いておくとワンクリックで開けて便利です（アイコンをDockにドラッグ）。

> ⚠️ 旧 `起動.command` はこのMacのターミナル環境ではエラー（forkpty）になるため使いません。`司令塔を起動.app` を使ってください。
> Codexを消す等でNodeが見つからない時は [nodejs.org](https://nodejs.org/ja) から無料インストールすれば動きます。

## パネル（できること）

| パネル | 状態 | 内容 |
| --- | --- | --- |
| 🍸 カクテル&ジン | ✅ 稼働中 | 世界のバー・蒸留メディアから自動収集→日本語訳→3行要約。上部に「今週の重要ニュース」。検索・カテゴリ対応（シンプル版）。 |
| 🔖 はてブ IT | ✅ 稼働中 | はてブIT人気エントリを巡回。**タップでブクマ（コメント）ページへ**。無料の公式RSS＋公開APIのみ。 |
| 🤖 Claude Code | ✅ 稼働中 | Hacker Newsで話題の「Claude Code」を巡回（人が投稿し議論がつくもの＝信頼の目安）。無料API。 |
| 🧠 ナレッジ&発想 | ✅ 稼働中 | 知識・哲学のメモを保存・振り返り＋毎日の「発想のタネ」。ローカル保存のみ。 |
| ⚡ 趣味の早見 | ✅ 稼働中 | カテゴリ別の早見データ（数値・用語・リンク）を自分で登録。ローカル保存のみ。 |

## ニュースパネルの使い方

- **最新を収集**：新しい記事を集めます（バックグラウンドで6時間ごとにも自動収集）。
- **翻訳を更新**：まだ日本語化されていない記事を翻訳します。
- **検索**：日本語訳・原文の両方からキーワードで探せます。
- **訳 / 原文**：表示を日本語要約か、英語原文かで切り替えます。
- **収集ソースの管理**：購読するRSS/AtomのURLを画面から追加できます。
- 各記事は **既読 / スター / 再翻訳** ができます。

## データの場所

収集結果とソース設定は `data/store.json` に保存されます。
（既読・スターもここに記録されます。）

## 手動でまとめて収集だけする

```bash
node server.mjs --collect-once
```

## デザイン

Bar Soutsu Tools の「ラボラトリー紙テーマ」（直角の枠・極小の大文字ラベル・ボタニカルグリーン）をベースにしつつ、
**個人専用の証として朱色の角印「私」＋ PRIVATE 表示**を加え、会社ツールとは一目で違うようにしています。
端末がダークモードなら自動で“夜のバー”配色に切り替わります。
※ 見た目を寄せているだけで、会社ツールの共通ナビ等には接続していません（完全に独立・非公開）。
